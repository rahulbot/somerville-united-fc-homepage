# Somerville United FC Homepage

Homepage for the Somerville United FC soccer organization. This is built as a SvelteKit app and deployed 
via a static compiliant to vanilla HTML/CSS/JS. Deployed and hosted via GH Pages.

## Developing

Make sure you have npm set up. Checkout the code and run `npm install` to get all the dependencies and such set up.
Then run `npm run dev` to run a local server on post 5001.

## Deploying

Just commit all your changes and push a semantically-formatted tag to GH (like `v2.1.12`). The actions will build 
a static site from the SvelteKit code, check that in, tag it, and deploy it. That only takes a minute.
